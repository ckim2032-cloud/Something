Mastering HTML Basics

HTML is the skeleton of every website. It uses "Tags" to tell the browser what to show.

1

The Anatomy of a Tag

Most HTML elements have an opening tag and a closing tag. The closing tag always has a forward slash /.

<h1>This is a big heading</h1>

<p>This is a simple paragraph.</p>

2

Text Elements

There are six levels of headings. h1 is the most important (and biggest), while h6 is the smallest.

Heading 1

Heading 2

Regular paragraph text looks like this.

<h1>Heading 1</h1>

<h2>Heading 2</h2>

<p>Regular paragraph...</p>

3

Links & Attributes

Attributes give tags extra information. For a link, we use the href attribute to say where the link goes.

<a href="https://google.com">Visit Google</a>

Tip: The "a" stands for "Anchor"!
